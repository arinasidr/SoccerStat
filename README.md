Запуск проекта

Проект состоит из 2 частей:
1) клиент
2) сервер

1. Запуск сервера
Перейдите в папку server: cd server
Запустите сервер: node index.js
После запуска сервер будет доступен по адресу:
http://localhost:3001

2. Запуск клиента
Перейдите в корневую папку проекта: cd ..
Запустите клиент: npm run dev
После запуска приложение будет доступно по адресу:
http://localhost:5173

Важно
Сервер должен быть запущен до запуска клиента
