Запуск проекта<br>
<br>
Проект состоит из 2 частей:<br>
1) клиент<br>
2) сервер<br>
<br>
1. Запуск сервера<br>
Перейдите в папку server: cd server<br>
Запустите сервер: node index.js<br>
После запуска сервер будет доступен по адресу:<br>
http://localhost:3001<br>
<br>
2. Запуск клиента<br>
Перейдите в корневую папку проекта: cd ..<br>
Запустите клиент: npm run dev<br>
После запуска приложение будет доступно по адресу:<br>
http://localhost:5173<br>
<br>
Важно!!<br>
Сервер должен быть запущен до запуска клиента<br>
